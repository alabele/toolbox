# toolbox

Personal developer automation scripts.

## Scripts

| Script | Description |
|--------|-------------|
| [claude-pr-loop](claude-pr-loop/) | Automated PR review-fix loop using Claude Code CLI. Drop a PR link or a batch of labeled PRs and let Claude review, fix, and push until clean. |

## Installation

Each script has its own README with setup instructions. The general pattern is:

```bash
ln -s ~/code/toolbox/<script-name>/<script-name> ~/.local/bin/<script-name>
```
